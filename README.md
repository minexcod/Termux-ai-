# Termux Assistant

A simple command-line assistant for Termux automation.

## Features
- Check current time and date
- Install packages
- Create and edit files
- Clear terminal
- Update system packages

## How to Run
python AI.py

## Commands
- time → current time
- date → current date
- install → install a package
- file → create or edit a file
- clr → clear screen
- update → update all packages
- edit ai → edit the assistant
- help → show all commands
- exit → quit
